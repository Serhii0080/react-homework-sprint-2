# Homework Sprint 2

This repository contains my React homework for Sprint 2.

## Technologies

- React
- TypeScript
- CSS Modules
- Create React App
- Yarn

## Getting Started

Install dependencies:

```bash
yarn install
```

Start the project:

```bash
yarn start
```

The app will be available at:

```text
http://localhost:3000
```

## Available Scripts

### `yarn start`

Runs the app in development mode.

### `yarn build`

Builds the app for production.

### `yarn test`

Runs tests in watch mode.
